## One-click install

**Install directly from Cursor.**

---

## Manual setup

Go to **Cursor Settings → MCP → Add new MCP Server**.

**Configuration:**

- **Name:** `playwright`
- **Type:** `command`
- **Command:** `npx @playwright/mcp@latest`

You can also verify the config or add arguments by clicking **Edit**.
