API referenceTest RunnerTestConfigOn this pageTestConfigPlaywright Test provides many options to configure how your tests are collected and executed, for example timeout or testDir. These options are described in the TestConfig object in the configuration file. This type describes format of the configuration file, to access resolved configuration parameters at run time use FullConfig. Playwright Test supports running multiple test projects at the same time. Project-specific options should be put to testConfig.projects, but top-level TestConfig can also define base options shared between all projects. playwright.config.tsimport { defineConfig } from '@playwright/test';export default defineConfig({ timeout: 30000, globalTimeout: 600000, reporter: 'list', testDir: './tests',}); Properties​ build​ Added in: v1.35 testConfig.build Playwright transpiler configuration.
