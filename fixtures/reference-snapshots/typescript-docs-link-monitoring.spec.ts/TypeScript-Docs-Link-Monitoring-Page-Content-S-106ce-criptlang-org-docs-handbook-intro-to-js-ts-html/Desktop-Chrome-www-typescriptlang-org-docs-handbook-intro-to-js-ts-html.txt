TypeScript with JavaScript This is when you use an editor which uses TypeScript to provide tooling like auto-complete, jump to symbol and refactoring tools like rename. The homepage has a list of editors which have TypeScript plugins. Providing Type Hints in JS via JSDoc In a .js file, types can often be inferred. When types can’t be inferred, they can be specified using JSDoc syntax. JSDoc annotations that come before a declaration will be used to set the type of that declaration. For example: js/** @type {number} */var x; x = 0; // OKx = false; // OK?!Try You can find the full list of supported JSDoc patterns in JSDoc Supported Types. @ts-check The last line of the previous code sample would raise an error in TypeScript, but it doesn’t by default in a JS project.
