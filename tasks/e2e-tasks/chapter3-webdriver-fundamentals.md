## Overview

Test fundamental browser interactions across a range of form elements, navigation controls, menus, mouse actions, and other core WebDriver capabilities.

---

## Web Form

- Verify the text input accepts typed text and the entered value is visible in the field.
- Verify the text input can be cleared and then retyped with new content.
- Verify the password field accepts input and is rendered as a password type (characters are masked).
- Verify the textarea accepts multi-line text, including content with line breaks, and retains it.
- Verify the disabled input field cannot be edited.
- Verify the read-only input field shows a pre-filled value that cannot be changed.
- Verify the dropdown (select element) lists the expected options and that selecting by visible label works correctly.
- Verify the dropdown also works when selecting by option value.
- Verify all expected options are present in the dropdown list.
- Verify the datalist input accepts a predefined suggestion from the list.
- Verify the datalist input also accepts a custom value that is not in the list.
- Verify checkboxes can be individually checked and unchecked.
- Verify radio buttons allow only one option to be selected at a time within a group.
- Verify the color picker input accepts a color value and reflects it.
- Verify the date picker accepts a date input and shows the selected date.
- Verify the range slider can be interacted with and returns a value within the expected min-max range.
- Verify the file upload input accepts a file selection.
- Verify the form can be submitted and the submitted values are displayed correctly on the confirmation page.

---

## Navigation

- Verify the Previous and Next links navigate between pages correctly.
- Verify clicking a numbered page link navigates to the correct page.
- Verify the active page indicator highlights the current page.
- Verify the first page does not have an active or clickable Previous link.
- Verify the last page does not have an active or clickable Next link.

---

## Dropdown Menu

- Verify left-clicking the dropdown button opens the expected list of menu items.
- Verify right-clicking the element opens a context menu dropdown.
- Verify double-clicking the element triggers the double-click action.

---

## Mouse Over

- Verify hovering the mouse over an element reveals hidden content or a tooltip.
- Verify the hover interaction counter increments each time the element is hovered.

---

## Drag and Drop

- Verify an element can be dragged from its source and dropped onto a target.
- Verify the drop target reflects the dragged item after the drop action completes.

---

## Draw in Canvas

- Verify the canvas element is present and visible on the page.
- Verify that drawing on the canvas produces visible marks (the canvas is no longer blank after interaction).

---

## Loading Images

- Verify that all images on the page load successfully and no broken image indicators are shown.

---

## Slow Calculator

- Verify the calculator display updates after a deliberate processing delay.
- Verify that basic arithmetic operations produce the correct result despite the delay.

---

## Index Page — Chapter 3 Links

- Verify all Chapter 3 feature links are visible on the index page.
- Verify clicking each link navigates to the correct page.
